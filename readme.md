
# Ignite Feed

A React application made using Vite with Typescript and CSS Modules.

## How to run

```bash
  yarn - To install all dependencies
  yarn dev - To run the local app server
```

## Web demo
https://ignite-feed-sable.vercel.app/

## Features

- Add a comment
- Delete a comment
- Like a comment multiple times
